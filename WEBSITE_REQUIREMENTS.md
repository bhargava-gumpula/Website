## Website Purpose

The purpose of this website is to showcase me. It should contain:

- All my CS projects
- Achievements
- My personal information
- A blog for updates (like a newsletter)
- Links to other things like GitHub, World Cube Association, etc.
- A way to register for classes I host:
  - Rubik's cubing
  - Python programming
  - C++ programming
- Contact options
- Class options:
  - Personal 1-on-1
  - Group classes
- Class locations:
  - Library (primary)
  - Garage (backup when library unavailable)
- Resources:
  - Rubik's cube algorithms
  - Python syllabus
  - Cubing syllabus

## Website Features

- AI chatbot to help with navigation and contacting
- About section for all my information
- Registration for classes
- Online payments for classes (bank account integration)
- Summer promotion banner: free group classes with promo code
- Class detail pages (per card): pricing by level/tier, course content, and syllabus
- Class calendar page: times, locations, and schedule
- Access restriction:
  - Rubik's cube algorithms should only be available with an account
- Website sign up / sign in
- Future membership support for premium content and personalized training
- Preferred hosting on Raspberry Pi
- Links to projects and external content (GitHub links, YouTube links, etc.)
- Easy blog publishing workflow (add a new blog post by providing information in the website)
- Testimonials section to build trust (student/client feedback and success stories)
- Class visibility control:
  - Mark a class as "coming soon"
  - Publish it later when ready
- Many website aspects should be customizable

## Website Styling Choices

- Minimalistic
- Easy access to different parts of the website
- Small interactive effects (for example, hover effects on tabs/clickables)
- Quick and easy class registration flow
- Fully responsive sizing for phone, laptop, TV, etc.
- Include graphics/icons (for example, Rubik's cube image, Python logo)
- Support adding a personal photo

## Long-Term Direction

- This is just the beginning
- The website should be extendable for future features
