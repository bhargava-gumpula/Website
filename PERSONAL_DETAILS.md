## Projects

### Web-based LED Controller (Raspberry Pi)

Built as part of Harvard CS50x. This is a web-based LED controller where users can control functions on an LED array. Users can control each LED individually and run pre-made patterns (for example circular spinning and random flashing).

- Demo: https://www.youtube.com/watch?v=UnIvxDrQVKg
- GitHub / project files: https://github.com/bhargava-gumpula/cs50/tree/main/project

### Online Grocery Store Implementation

Built as part of Harvard CS50P. This project includes an account system with multiple users, sign in, username/password updates, product catalog browsing, cart add/remove, and checkout with card payment. Transactions are stored in a file. Users start with a balance of $100 and cannot purchase with insufficient funds.

- Demo: https://www.youtube.com/watch?v=BkqTLnhLsCg
